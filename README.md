# FutMatch - Sistema de Peladas Anônimas

O FutMatch é uma plataforma que permite que jogadores encontrem e participem de peladas próximas a eles. O sistema oferece funcionalidades como criação de salas, busca de partidas, sistema de ranking e feedback.

## Funcionalidades Principais

- ✅ Criação de salas personalizadas
- ✅ Busca de partidas por localização
- ✅ Sistema de feedback e avaliação
- ✅ Filtros por nível de habilidade
- ✅ Perfil do jogador com estatísticas
- ✅ Sistema de ranking

## Tecnologias Utilizadas

- React 18
- TypeScript
- Material-UI
- React Router
- Vite
- Date-fns

## Pré-requisitos

- Node.js 16 ou superior
- npm ou yarn

## Instalação

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/futmatch-frontend.git
cd futmatch-frontend
```

2. Instale as dependências:
```bash
npm install
# ou
yarn install
```

3. Inicie o servidor de desenvolvimento:
```bash
npm run dev
# ou
yarn dev
```

4. Acesse a aplicação em `http://localhost:5173`

## Estrutura do Projeto

```
src/
  ├── components/     # Componentes reutilizáveis
  ├── pages/         # Páginas da aplicação
  ├── theme/         # Configuração do tema
  ├── types/         # Definições de tipos TypeScript
  ├── utils/         # Funções utilitárias
  ├── store/         # Estado global (Redux)
  ├── hooks/         # Hooks personalizados
  ├── services/      # Serviços e APIs
  └── assets/        # Recursos estáticos
```
